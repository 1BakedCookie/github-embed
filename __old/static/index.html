Github embed tool

If you want to share some code snippet on your website but:

  • You want to get it always up to date with your code on Github.
  • You don't like to use gists.
  • You want to show few code snippets per once.
You've found the right place!

How to get started?

The first step you need to do is to create .gh-embed.json somewhere in your repo (usually in root). This file means embedding settings file and it must contain valid JSON.

The second step is fillling .gh-embed.json by settings described below.

The third step is adding an iframe on your webpage. "src" attribute of your iframe should be formatted as: //gh-embed.matreshka.io/v0/{OWNER}/{REPO}/{PATH_TO_SETTINGS}/?ref={REF}, where OWNER is repo owner, REPO is target repository, PATH_TO_SETTINGS is a path to .gh-embed.json (usually root, which means PATH_TO_SETTINGS is optional), REF is specific branch or commit (master by default). v0 means the version of embedder. Example:

<iframe
width="100%"
height="400"
frameborder="0"
src="//gh-embed.matreshka.io/v0/matreshkajs/matreshka_examples/treeview/?ref=gh-pages"
></iframe>

Settings (.gh-embed.json)

  • linenums: BOOLEAN - show line numbers or not
  • embed: ARRAY - an array of embedded files
{
	"linenum": true,
	"embed": [...]
}

embed item

  • path: STRING - a path to embedded file relative to root of the repo (required).
  • type: STRING - a type of the file you need to embed (eg language). This is optional, the type can be calculated automatically from source file name.
  • label: STRING - what to display in navigation. By default it's a name of embedded file
  • active: BOOLEAN - is shippet active

In case if you'd like to embed a file from external repo there are some extra options

  • owner: STRING - an owner of a repo (user's nickname)
  • repo: STRING - repository name.
  • ref: STRING - branch or commit

There is one more thing: you can add to your embedding list any webpage. It could be useful if you're web developer and you'd like to show how does your code work. You need to set type option as "htmlpage" and assign weppagee URL to "url" property

Example

{
	"linenums": true,
	"embed": [{
		"type": "htmlpage",
		"url": "//matreshkajs.github.io/matreshka_examples/treeview/"
	}, {
		"type": "js",
		"label": "app.js",
		"path": "treeview/js/app.js"
	}, {
		"type": "js",
		"label": "tree.class.js",
		"path": "treeview/js/tree.class.js"
	}, {
		"type": "js",
		"label": "tree-leaf.class.js",
		"path": "treeview/js/tree-leaf.class.js"
	}]
}
